# meeting_place
Проект «Место встречи» представляет собой актуальное и социально значимое решение, направленное на оптимизацию процесса планирования групповых встреч в условиях современного города. Разработанная математическая модель и алгоритм позволяют не только минимизировать временные затраты участников, но и обеспечить справедливое распределение условий.
